# BankAPI
## Установка:
-git clone 

-mvn install

-java -jar BankAPI.jar

## База данных:
H2 Embedded

### Параметры подключения к БД:
Path: путь до папки, где будут храниться файлы БД

Host: localhost

Port: 9092

User: svn

Password: 1234

## Логирование:
В файле logback.xml нужно указать путь по файла, куда будут писаться логи
